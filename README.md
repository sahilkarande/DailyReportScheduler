# Daily Report Scheduler

A lightweight Android app to quickly create and manage structured daily reports.

![App Screenshot](app_image.jpeg)

## Features

* Create daily report with predefined time slots
* Auto-generate formatted report text
* Copy to clipboard
* Share via WhatsApp, Email, etc.
* Simple and clean UI (Jetpack Compose)

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3

## Usage

1. Select batch and date
2. Enter schedules
3. Generate report
4. Copy or share instantly
